# Template de Quadro Kanban

Use este template para criar seu quadro Kanban físico ou digital.

## Colunas Sugeridas

1. **A Fazer**: Tarefas pendentes.
2. **Fazendo**: Trabalho em progresso (limite: 3 itens).
3. **Revisão**: Verificação de qualidade.
4. **Feito**: Tarefas concluídas.

## Exemplo de Cartões

- Cartão 1: Contatar doadores para arrecadação.
- Cartão 2: Organizar distribuição de alimentos.
- Cartão 3: Avaliar necessidades das famílias.

Adapte conforme seu projeto social.
