# Guia de Planejamento de Sprint

Planeje o Sprint selecionando itens do Backlog.

## Passos

1. **Revisar Objetivo do Sprint**: Definir meta.
2. **Selecionar Itens**: Escolher tarefas viáveis.
3. **Estimativa**: Discutir esforço.
4. **Compromisso**: Time se compromete com as tarefas.

## Dicas

- Sprint de 2 semanas para projetos sociais.
- Foque em valor para os assistidos.
